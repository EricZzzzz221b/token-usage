# 更新记录

## v1.2.0-beta.14 · 2026-07-22

- 新增 Codex 实时任务面板，分别显示多个进行中任务及思考、执行、等待、完成、失败和中断状态。
- 最近完成扩展至 5 条，并支持点击后直接打开对应 Codex 会话。
- 状态栏新增任务状态、运行时间和活动任务数，任务完成时可发送系统通知。
- 自动识别订阅与 API 模式；API 模式下展示 Credits，订阅模式下保持额度窗口为主要信息。
- 新增官方使用限额重置机会、可用次数及到期信息，详情默认折叠并记住当前展开状态。
- 7 天窗口使用日期显示重置时间，macOS 新增 Dock 图标显示开关。
- 取消通透强度控制，统一为更稳定、清晰的标准外观。
- 会话标题改为读取 Codex 侧边栏标题，并过滤插件调用文本和错误兜底标题。
- 任务扫描改为共享快照与 JSONL 增量读取，避免重复解析大型会话日志。

## v1.1.5 · 2026-07-16

- 用量接口限流或暂时不可用时采用有上限的指数退避，避免频繁重试。
- 设置改为串行保存，快速连续操作不会互相覆盖。
- 设置保存失败时恢复上一次有效值，并在界面中提示。
- Windows 发布版不再额外显示控制台窗口。

## v1.1.4 · 2026-07-13

- 按照 Apple 的 `NSGlassEffectView.contentView` 结构重做 macOS 液态玻璃内容层。
- 浮窗文字、卡片和控件可根据玻璃材质的局部外观自动调整深浅配色。
- 不读取桌面壁纸、不截取其他应用内容，也不需要屏幕录制权限。
- 增加背景变化抗抖动，避免临界明暗区域频繁闪烁。

## Windows v1.0.0 · 2026-07-13

- 首个 Windows 正式版本，支持 Windows 11 x64 和 Windows 10 22H2 x64。
- 提供 MSI 和 EXE 两种安装包，并附带 SHA-256 校验文件。
- 支持系统托盘、标准/紧凑浮窗、Mica/Acrylic、自动刷新、通知和窗口设置。
- 安装包和完整说明已发布到 [Windows v1.0.0 Release](https://github.com/EricZzzzz221b/token-usage/releases/tag/windows-v1.0.0)。

## v1.1.3 · 2026-07-13

- 根据浮窗所在位置的桌面壁纸自动选择深色或浅色文字。
- 移动浮窗后自动更新文字对比度，不需要手动调整。

## v1.1.2 · 2026-07-13

- 增加深色和浅色文字选项，改善深色背景下的可读性。

## v1.1.1 · 2026-07-13
