# Token用量 v1.2.0

- 新增 Codex 实时任务状态、多任务分别展示和完成通知。
- 最近完成保留 5 条，并支持一键打开对应 Codex 会话。
- 自动识别订阅/API 模式；API 模式按官方数据展示 Credits。
- 新增官方使用限额重置机会、次数和到期信息。
- 状态栏显示任务状态、运行时间、活动任务数与剩余额度。
- 7 天窗口使用日期显示重置时间，macOS 新增 Dock 图标开关。
- 任务扫描改为共享快照与 JSONL 增量读取，减少重复后台工作。

适用于 macOS 13+ Apple Silicon。当前安装包采用 ad-hoc 签名，首次启动请在 Finder 中右键应用并选择“打开”。
