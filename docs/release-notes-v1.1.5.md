# Token用量 v1.1.5

发布日期：2026-07-16

## 本次更新

- 用量接口限流或暂时不可用时采用有上限的指数退避，减少无效请求并提高恢复稳定性。
- 设置改为串行保存，快速连续调整通知、刷新周期或托盘显示周期时不会互相覆盖。
- 设置、浮窗偏好或登录时启动保存失败时，自动恢复上一次有效值并显示提示。
- Windows 发布构建不再额外显示控制台窗口。
- 增加退避、连续设置写入和保存失败回滚的自动化测试。

## 安装

1. 下载并打开 `TokenUsage_1.1.5_arm64.dmg`。
2. 将 `Token用量.app` 拖入 `Applications`。
3. 首次打开时，在 Finder 中右键应用并选择“打开”，然后再次确认。

系统要求：Apple Silicon Mac，macOS 13 或更高版本。

## 隐私

OAuth 凭据仍然只在本机内存中读取，只用于请求官方用量接口。本版本不包含遥测、屏幕捕获或行为追踪。
