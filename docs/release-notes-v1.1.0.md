# Token用量 v1.1.0

发布日期：2026-07-12

## 主要变化

- 用量显示改为“剩余额度”倒扣逻辑：满额为 100%，使用后逐步下降，用尽为 0%。
- 紧凑模式、详细模式、进度条、辅助功能标签和状态栏百分比全部统一为剩余额度。
- 风险颜色反转：剩余 30% 及以下为黄色，10% 及以下为红色，0% 为用尽状态。
- 通知文案改为剩余 30%、剩余 10% 和额度用尽。
- 状态栏详情改为显示各周期剩余额度。
- 延续 v1.0 的新应用图标、独立状态栏模板图标和快捷模式切换。

## 兼容性

- Apple Silicon Mac
- macOS 13 或更高版本
