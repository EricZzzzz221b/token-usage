# Token用量 v1.2.0

发布日期：2026-07-22

## 这次更新解决什么问题

以前查看 Codex 额度或确认任务是否完成，需要不断切回应用。v1.2.0 将用量和任务进度放进同一个常驻浮窗：离开 Codex 后，仍能从状态栏看到任务进度，并在完成后快速返回原会话。

## 新功能

- 实时任务状态：思考中、执行中、需要操作、已完成、失败和已中断。
- 多任务分别展示，不再把多个进行中的任务合并为一条。
- 最近完成保留 5 条，支持一键打开对应 Codex 会话。
- 状态栏显示任务运行时间、活动任务数与所选额度窗口。
- 自动识别订阅/API 模式；API 模式按官方数据展示 Credits。
- 展示官方返回的使用限额重置机会、次数与到期信息。
- macOS 可选择是否在 Dock 中显示应用图标。

## 体验调整

- 详细模式恢复以用量信息为主的层级，并新增“用量/任务”切换。
- 紧凑模式保持单行结构，同时显示任务状态、运行时间和额度。
- 使用限额重置默认折叠，切换标签或模式时不会意外改变展开状态。
- 7 天窗口重置时间改为日期。
- 移除通透强度滑块，统一使用标准外观。

## 性能与可靠性

- 任务状态快照由前端与状态栏共享，不再重复扫描会话。
- 会话 JSONL 使用字节游标增量读取，仅解析新增事件。
- 侧边栏标题与 SQLite 标题缓存独立更新，不再让标题变化触发日志重读。
- 文件被截断或替换时自动重建解析状态。

## 隐私

任务功能只在本机读取 Codex 会话生命周期事件。应用不会上传或另行保存完整提示词、回答、命令输出和文件内容，也不会记录 Access Token、Refresh Token 或原始认证文件。

## 安装

本版本面向 macOS 13+ Apple Silicon。退出旧版后打开 DMG，将 `Token用量.app` 拖入 `Applications`。由于当前包未经过 Apple 公证，首次启动请在 Finder 中右键应用并选择“打开”。
