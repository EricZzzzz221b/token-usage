# Token用量 v1.2.1

v1.2.1 是任务跳转与使用限额重置信息的稳定性更新。

## 本次更新

- 进行中的任务卡片支持点击，直接打开对应 Codex 会话；同时支持键盘聚焦与打开。
- 修复启动阶段任务快照与事件订阅之间的竞态，避免“最近完成”短暂为空。
- 兼容使用限额重置接口的直接数组、嵌套对象及不同字段命名。
- 官方接口某次刷新偶发缺少重置机会时，在 30 分钟内沿用最近一次有效数据；官方明确返回 0 次时仍立即隐藏。
- 只有可用次数、没有机会明细时显示可靠摘要，不再提供点开后无内容的折叠按钮。
- 使用固定尺寸、单一图形旋转的折叠箭头，消除展开与收起时的跳动。
- README 产品截图改为标准 sRGB PNG，改善 GitHub 深色与浅色页面中的显示一致性。

## 安装

适用于 Apple Silicon Mac，要求 macOS 13 或更高版本。退出旧版后打开 DMG，将 `Token用量.app` 拖入 `Applications`。

当前安装包采用 ad-hoc 签名且未经过 Apple 公证。首次启动时，请在 Finder 中右键应用，选择“打开”，并再次确认。

## 隐私

应用只在本机读取 Codex OAuth 登录状态、官方用量响应和会话生命周期事件，不上传凭据或完整会话内容，也不包含遥测。
