# Token用量 v1.1.1

发布日期：2026-07-13

## 变化

- 所有按钮与窗口拖动区域保持系统默认箭头鼠标，不再显示点击手型、抓取或等待光标。
- 设置页新增“状态栏显示周期”，可选择显示 5 小时窗口或 7 天窗口的剩余额度。
- 状态栏周期选择保存后立即生效，并在重启后保持。
- 旧版设置自动迁移，默认显示 5 小时窗口。

## 兼容性

- Apple Silicon Mac
- macOS 13 或更高版本
