# Token用量 v1.1.2

发布日期：2026-07-13

## 修复

- 新增“文字颜色”设置：自动（跟随系统）、深色文字、浅色文字。
- 修复系统为浅色模式但浮窗位于深色或黑色壁纸上时，深色文字与背景混在一起的问题。
- 文字颜色选择会同步应用到紧凑模式、详细模式和设置页，并在重启后保持。
- 使用 CSS 语义颜色变量统一标题、数值、辅助文字、按钮、分隔线、卡片和进度轨道的对比度。
- 旧版窗口设置自动迁移为“自动（跟随系统）”。

## 使用建议

当系统为浅色模式、浮窗背景较暗时，在设置 > 外观 > 文字颜色中选择“浅色文字”。

## 兼容性

- Apple Silicon Mac
- macOS 13 或更高版本
