# Token用量 v1.0.0

发布日期：2026-07-12

这是 Token用量的首个可安装版本。

## 功能

- 在 macOS 状态栏查看 Codex 用量。
- 标准和紧凑两种桌面浮窗。
- 显示 5 小时与 7 天窗口、重置时间和刷新状态。
- 支持自动刷新、余额提醒和登录时启动。
- 支持始终置顶、位置锁定和鼠标穿透。
- 使用 macOS 原生液态玻璃效果，并提供连续的玻璃效果调节。
- 提供独立的应用图标和状态栏图标。

## 安装

下载 `TokenUsage_1.0.0_arm64.dmg`，打开后将 `Token用量.app` 拖入 `Applications`。

当前安装包尚未经过 Apple 公证。首次启动时，请在 Finder 中右键应用并选择“打开”。

系统要求：Apple Silicon Mac，macOS 13 或更高版本。
